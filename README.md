# My Simple React Blog

This is a  project built with React. It shows how to use **components** and **props** to display a small blog site.

## Features

- Header showing the blog name
- About section with a logo and description
- List of articles from a data file

## How to Run

1. Clone this repository:

```bash
git clone <your-repo-url>
cd <repo-folder>

Install dependencies:

npm install


Start the app:

npm run dev   # for Vite
# or
npm start     # for Create React App


## Notes

All content comes from blog.js

Data is passed to components using props
